# Overview

I am building small, focused programs to strengthen my foundation as a software
engineer and to sharpen the daily workflow I rely on: writing clean code, using
version control, documenting clearly, and demonstrating my work.

This program is a minimal Node.js application written in JavaScript. It builds a
greeting message through a small reusable function and prints `Hello, World!` to
the console when it is run with `node index.js`.

My purpose for creating it was to establish and verify a reliable end-to-end
development loop — edit in VS Code, run locally, commit with Git, and publish to a
public GitHub repository — so I can move quickly and confidently on larger projects
later in my learning.

[Software Demo Video](http://youtube.link.goes.here)

# Development Environment

I developed this software using Visual Studio Code as my editor and Git with GitHub
for version control. I ran and tested the program locally with the Node.js runtime.

The program is written in JavaScript (ES2020+) and executed with Node.js. It uses
only the standard `console` API, with no external dependencies.
