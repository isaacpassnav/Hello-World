# Overview

This is my introductory "Hello World" program for CSE 310 – Applied Programming.
The purpose of this project is to set up and verify my development workflow for the
course: writing code, using Git and GitHub, documenting with Markdown, and recording
a demonstration video.

The software is a small Node.js program written in JavaScript. It builds a greeting
message and prints `Hello, World!` to the console. Although it is simple, it exercises
the full loop I will use all term: edit code in VS Code, run it, commit, and push to
a public GitHub repository.

[Software Demo Video](https://youtu.be/REEMPLAZAR-CON-TU-LINK)

# Development Environment

- **Editor:** Visual Studio Code
- **Language:** JavaScript (ES2020+)
- **Runtime:** Node.js
- **Version control:** Git + GitHub
